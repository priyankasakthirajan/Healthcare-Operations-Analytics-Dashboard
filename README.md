# 🏥 Healthcare Operations Analytics Dashboard

## 📌 Project Overview

This project showcases an end-to-end **Healthcare Operations Analytics Dashboard** developed using **Microsoft Excel** and **Power BI**. The dashboard transforms raw healthcare data into interactive visualizations and actionable insights, enabling healthcare organizations to monitor operational performance and support data-driven decision-making.

---

## 🎯 Objective

To analyze hospital operational data and provide meaningful insights into patient admissions, billing, department performance, doctor performance, and overall healthcare efficiency through interactive dashboards.

---

## 🛠️ Tools & Technologies

- Microsoft Excel
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

---

## 📊 Dashboard Features

- Executive Summary Dashboard
- Patient Demographics Analysis
- Department Performance Dashboard
- Doctor Performance Analysis
- Revenue & Billing Analysis
- Monthly Admission Trends
- Average Length of Stay
- Gender & Age Group Distribution
- Interactive Filters & Slicers
- KPI Cards

---

## 📈 Key Performance Indicators (KPIs)

- Total Patients
- Total Admissions
- Total Revenue
- Average Bill Amount
- Average Length of Stay
- Department-wise Admissions
- Doctor-wise Patient Count
- Payment Method Analysis

---

## 📂 Dataset

The project uses a sample healthcare dataset containing:

- Patients

- Doctors

- Billing Information

The dataset was created for learning and portfolio purposes.
